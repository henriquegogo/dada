# [THIS IS EXTREMELY PRE-ALPHA-DEV-STAGE]

## What is?
Is a SQLite abstraction to use as a REST API with zero config.

## Routes
Return all tables
```
/
```

Get all items all from table
```
/table
```

Get a specific item from table
```
/table/item_id
```

## Next steps
