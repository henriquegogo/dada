# Da-Da-Da (Database API)

## What is?
Is a database abstraction to use as a REST API with zero config.

## How to install and use
(Not published yet)
```
pip install dadada
```

## Routes
Return all data
```
GET /
```

Get a specific item
```
GET /item_id
```

Create an item
```
POST /item_id
DATA 'Data'
```

Update an item
```
PUT /item_id
DATA 'Data'
```

Delete an item
```
DELETE /item_id
```
